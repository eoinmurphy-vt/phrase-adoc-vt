== AWS ECR - Rôles IAM

Lorsque les conteneurs {product-name} sont déployés dans AWS et qu'une instance EC2 se voit attribuer le rôle &quot;`+EC2 Container Registry+`&quot; Read Access, le registre ECR d'AWS peut être analysé sans clé d'accès ni clé secrète.

Voici comment créer un rôle AWS et l'attribuer au nœud.

=== Sélectionner l'instance

Notez que le rôle IAM est soit vide, soit n'inclut pas le rôle ECR.

image:ecr1.png[awsrole]

=== Attacher un rôle

Sélectionnez Actions -> Instance Settings -> Attach/Replace IAM Role

image:ecr2.png[awsrole]

Si vous n'avez pas encore créé le rôle ECR, cliquez sur Créer un nouveau rôle IAM. Saisissez le nom du rôle.

image:ecr3.png[rôle d'attachement]
