== Configuration et mise en place

=== Obtenir l'application

==== GitHub

Téléchargez le dernier tarball de l'application (`neuvector_app.tar.gz`) depuis le https://github.com/neuvector/neuvector-splunk-app[dépôt neuvector/neuvector-splunk-app].

==== Splunkbase

Téléchargez le dernier fichier tarball de l'application depuis https://splunkbase.splunk.com/apps[Splunkbase].

==== Navigateur d'applications Splunk

Dans l'interface utilisateur de Splunk, cliquez sur le menu déroulant Applications, cliquez sur "Trouver plus d'applications", puis recherchez {product-name} l'application Splunk.

=== Installation et configuration

Installez l'application en téléchargeant le tarball ou en suivant les invites de Splunkbase.

==== Configurez syslog dans la console {product-name}

Allez dans Paramètres -> Configuration -> Syslog

[loweralpha]
. définissez la valeur du serveur comme l'adresse IP sur laquelle Splunk fonctionne
. choisissez TCP comme protocole
. définissez le numéro de port sur 10514
. choisissez le niveau d'information
. cliquez sur SOUMETTRE pour enregistrer le paramètre.

image:syslog-config.png[syslog]

Vous pouvez configurer plusieurs clusters pour envoyer des syslogs à votre instance Splunk et votre instance Splunk recevra ces syslogs en temps réel.
