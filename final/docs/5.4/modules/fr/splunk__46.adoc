== Mise en place et configuration

=== Obtenir l'application

==== GitHub

Téléchargez la dernière version de l'application (`neuvector_app.tar.gz`) depuis le https://github.com/neuvector/neuvector-splunk-app[dépôt]

==== Splunkbase

Téléchargez la dernière version de l'application à partir de https://splunkbase.splunk.com/apps[Splunkbase].

==== Navigateur Splunk Apps

Dans l'interface Splunk, cliquez sur le menu déroulant Apps, cliquez sur "Find More Apps", puis recherchez {product-name} Splunk App.

=== Installation et configuration

Installez l'application en téléchargeant l'archive ou en suivant les instructions de Splunkbase.

==== Configurer syslog dans la console {product-name} 

Allez dans Settings -> Configuration -> Syslog

[loweralpha]
. définir la valeur du serveur comme l'adresse IP de Splunk
. choisir le protocole TCP
. fixer le numéro de port à 10514
. choisir le niveau d'information
. cliquez sur SUBMIT pour enregistrer le paramètre.

image:syslog-config.png[syslog]

Vous pouvez configurer plusieurs clusters pour qu'ils envoient des syslogs à votre instance splunk et votre instance splunk recevra ces syslogs en temps réel.
