== Einrichtung und Konfiguration

=== Die App erhalten

==== GitHub

Laden Sie das neueste App-Tarball (`neuvector_app.tar.gz`) aus dem https://github.com/neuvector/neuvector-splunk-app[neuvector/neuvector-splunk-app Repository] herunter.

==== Splunkbase

Laden Sie das neueste App-Tarball von https://splunkbase.splunk.com/apps[Splunkbase] herunter.

==== Splunk Apps-Browser

Klicken Sie in der Splunk-Benutzeroberfläche auf das Dropdown-Menü Apps, klicken Sie auf "Weitere Apps finden" und suchen Sie nach {product-name} Splunk App.

=== Installation und Einrichtung

Installieren Sie die App, indem Sie entweder das Tarball hochladen oder den Aufforderungen von Splunkbase folgen.

==== Konfigurieren Sie Syslog in der {product-name} Konsole

Gehen Sie zu Einstellungen -> Konfiguration -> Syslog

[loweralpha]
. Setzen Sie den Serverwert auf die IP-Adresse, auf der Splunk läuft
. Wählen Sie TCP als Protokoll
. Setzen Sie die Portnummer auf 10514
. Wählen Sie die Informationsstufe
. Klicken Sie auf ABSENDEN, um die Einstellung zu speichern.

image:syslog-config.png[syslog]

Sie können mehrere Cluster konfigurieren, um Syslog an Ihre Splunk-Instanz zu senden, und Ihre Splunk-Instanz wird diese Syslogs in Echtzeit empfangen.
