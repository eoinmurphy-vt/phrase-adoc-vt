== Einrichtung und Konfiguration

=== Die App bekommen

==== GitHub

Laden Sie den neuesten App-Tarball (`neuvector_app.tar.gz`) aus dem https://github.com/neuvector/neuvector-splunk-app[neuvector/neuvector-splunk-app Repository] herunter.

==== Splunkbase

Laden Sie den neuesten Tarball der Anwendung von https://splunkbase.splunk.com/apps[Splunkbase] herunter.

==== Splunk Apps Browser

Klicken Sie in der Splunk-Benutzeroberfläche auf das Dropdown-Menü "Apps", klicken Sie auf "Weitere Apps suchen" und suchen Sie dann nach {product-name} Splunk App.

=== Installation und Einrichtung

Installieren Sie die Anwendung, indem Sie entweder den Tarball hochladen oder den Aufforderungen von Splunkbase folgen.

==== Konfigurieren Sie syslog in der Konsole {product-name} 

Gehen Sie zu Einstellungen -> Konfiguration -> Syslog

[loweralpha]
. Setzen Sie den Serverwert auf die IP-Adresse, unter der Splunk läuft
. TCP als Protokoll wählen
. Portnummer auf 10514 setzen
. Info-Level wählen
. Klicken Sie auf SUBMIT, um die Einstellung zu speichern.

image:syslog-config.png[syslog]

Sie können mehrere Cluster so konfigurieren, dass sie Syslog an Ihre Splunk-Instanz senden, und Ihre Splunk-Instanz wird diese Syslogs in Echtzeit empfangen.
